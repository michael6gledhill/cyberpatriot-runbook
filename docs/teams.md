# Teams & Requests

## Create Team (Coach/Admin)
- Open the Team Management/Coach dashboard → Create New Team.
- Enter Name, Team ID (NN-NNNN), Division.
- The creator is stored as `created_by_user_id` automatically.

## Join Existing Team
- Enter Team ID → send join request to the team creator.
- Duplicate requests are prevented.

## Edit Team (Admin)
- Select a team in Admin dashboard → Edit Team → change name/division.

## Delete Team (Admin)
- Select a team → Delete; removes team and members (use carefully).

## Approval Flow
- Pending users/requests appear in dashboards for approval/rejection.
